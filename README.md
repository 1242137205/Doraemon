`Doraemon` 目标是打造一个AI百宝箱，帮助你在电脑上快速的启动各种AI应用，免去各种环境配置的烦恼，更好、更高效率的使用AI。

### 优点

- 安全-代码全部透明开源
- 轻量-安装包体积轻量可控
- 极速-使用国内镜像源解决网络缓慢问题
- 干净-每个项目独立环境安装，不会污染系统环境

### 开发

```bash
$ npm i
$ npm run start
$ npm run package
```
