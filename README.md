### 优点

- 安全-代码全部透明开源
- 轻量-启动器大小150M左右
- 极速-使用国内镜像源解决网络缓慢问题
- 干净-每个项目独立环境安装，不会污染系统环境

### 开发

```bash
$ npm i
$ npm run start
$ npm run package
```
